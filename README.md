# OmniForge Academy

Premium fan-made landing page inspired by the Ben 10 universe.

## Teknoloji

- Next.js
- React
- Tailwind CSS
- Framer Motion

## Kurulum

```bash
npm install
npm run dev
```

## Proje Özellikleri

- Tam responsive, premium cartoon-stil
- Bölümler, kampüs yaşamı, başvuru süreci, öğrenci yorumları ve SSS
- Reusable component yapısı ve içerik veri modelleri
- SEO meta, semantik HTML ve kontrollü motion animasyonları
